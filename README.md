# Blue Archive Helper

适用于 AstrBot 的蔚蓝档案活动助手插件，提供：

- 活动查询（进行中、即将开始分开查询）
- 活动结束前主动提醒（可按群开启/关闭）
- 总力战数据查询（各难度最低排名、档线）

## 功能说明

### 1) 活动查询

- `/ba进行中`：查询当前进行中的活动
- `/ba即将开始`：查询“即将开始”窗口内的活动
- `/ba活动`：显示查询入口提示

### 2) 活动结束提醒

- `/ba提醒开启 [分钟]`：开启本群提醒，支持自定义提前分钟数
- `/ba提醒关闭`：关闭本群提醒
- `/ba提醒状态`：查看本群提醒状态与当前生效配置

提醒规则：

- 轮询周期内，命中提醒窗口的活动会合并为一条消息发送
- 同一活动会做去重，避免在后续轮询重复刷屏
- 每个群独立订阅，不互相影响

### 3) 总力战数据查询

- `/ba总力战各难度最低排名 [服务器]`：查询最新一期总力战各难度的最低排名
- `/ba总力战档线 [服务器]`：查询最新一期总力战各档线分数

服务器参数可选：`官服`（默认）、`B服`、`日服`

### 4) 总力战助战查询

- `/ba助战`：显示助战查询帮助
- `/ba助战 [角色名] [筛选参数...]`：查询符合条件的高练度助战

**命令格式：**

```
/ba助战 [服务器] [角色名] [星级] [技能1] [技能2] [技能3] [大招] [专武] [等级] [好友] [排序] [页]
```

**筛选参数（可选，默认值）：**

| 参数  | 说明       | 默认值 |
| ----- | ---------- | ------ |
| 星级  | 学生星级≥  | 5      |
| 技能1 | 技能1等级≥ | 10     |
| 技能2 | 技能2等级≥ | 10     |
| 技能3 | 技能3等级≥ | 10     |
| 大招  | 大招等级≥  | 5      |
| 专武  | 专武星级≥  | 3      |
| 等级  | 学生等级≥  | 90     |
| 好友  | 好友数量≤  | 49     |

**排序参数（可选）：**

| 值  | 说明         |
| --- | ------------ |
| 0   | 默认         |
| 1   | 好感度降序   |
| 2   | 好感度升序   |
| 3   | 好友数量降序 |
| 4   | 好友数量升序 |

## 配置说明

插件支持 WebUI 配置，配置文件为 `_conf_schema.json`。

| 配置项                   | 类型  | 默认值 | 说明                                   |
| ------------------------ | ----- | -----: | -------------------------------------- |
| `api_timeout_seconds`    | float | `10.0` | 活动接口请求超时时间（秒）             |
| `upcoming_window_hours`  | int   |   `24` | “即将开始”活动查询窗口（小时）         |
| `reminder_poll_minutes`  | int   |   `30` | 结束提醒轮询周期（分钟）               |
| `default_remind_minutes` | int   |   `30` | `/ba提醒开启` 不带参数时的默认提前分钟 |
| `max_remind_minutes`     | int   | `1440` | 允许设置的最大提前提醒分钟数           |

## 注意事项

- 本插件依赖外部活动接口，接口异常时会提示”活动数据获取失败，请稍后重试”。
- 主动提醒依赖平台支持主动消息能力。
- 提醒轮询不是秒级触发，建议根据群需求设置合理的 `reminder_poll_minutes`。
- 总力战数据来源于 [arona.icu](https://arona.icu)，使用时请备注数据来源。
